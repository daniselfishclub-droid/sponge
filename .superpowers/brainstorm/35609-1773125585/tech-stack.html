기술 스택 & 구현 방향

초안(MVP)을 빠르게 만들기 위한 추천 구성

추천

Next.js + Tailwind CSS + Supabase

빠른 개발 + 확장성 모두 확보

  • Next.js (App Router) — SSG로 빠른 로딩, SEO 최적화
  • Tailwind CSS — 다크 테마 구현 용이, 빠른 스타일링
  • Supabase — 이미 사용 중인 DB. 신청 폼, 후기 관리에 활용
  • Vercel 배포 — Next.js 최적 호스팅, 무료 티어로 시작

장점

  • Claude Code로 빠르게 개발 가능
  • 커뮤니티 성장 시 기능 확장 용이
  • Supabase 연동으로 데이터 일원화

고려사항

  • React 기반이라 초기 세팅 필요
B

정적 사이트 (HTML + CSS + JS)

가장 단순한 접근. 순수 정적 페이지로 배포

  • HTML/CSS/JS 직접 작성
  • Netlify 또는 GitHub Pages 배포
  • 신청 폼은 Google Forms 또는 Tally 임베드

장점

  • 빌드 과정 없이 바로 배포
  • 가장 빠른 초기 구현

단점

  • 페이지 추가/수정 시 중복 코드
  • 동적 기능 확장 어려움
C

노코드 (Framer / Webflow)

코드 없이 드래그 앤 드롭으로 제작

  • Framer 또는 Webflow로 비주얼 빌딩
  • CMS 기능으로 후기/데모데이 관리
  • 폼 기능 내장

장점

  • 디자인 자유도 높음
  • 비개발자도 수정 가능

단점

  • 월 비용 발생 ($15-40/월)
  • Supabase 연동 복잡
  • 커스텀 기능 제한

추천 이유: Next.js + Tailwind + Supabase

기획서에 Supabase를 이미 활용 중이고, Claude Code로 개발 가능하므로 Next.js가 최적입니다. 초기에는 정적 페이지처럼 심플하게 만들되, 이후 신청 폼·후기 관리·포인트 시스템 등 동적 기능을 자연스럽게 확장할 수 있습니다.
